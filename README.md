# Install

```
// install all dependence

npm install

// configurate config/server.config.js
// make sure you have run the database mongodb and redis

npm run start
```

# Framework

Front-End (webpack)
* React
* Redux
* React-Router
* Flat-UI

Back-End
* Koa2

Database
* MongoDB
* Redis
